# Pokemon-App

## SETUP
npm install
=======

Clone the project 

```bash
  https://github.com/Lrfith/Pokemon-App.git
```
You wll get `Pokemon-App` folder.
Go to the project directory

```bash
  cd Pokemon-App
```
Install dependencies

```bash
  npm install
```
Start the App in Android Simulator
```bash
  npm run android
```
